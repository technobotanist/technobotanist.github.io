- Produced Unity toy scenes, music loops, and documentation highlighting asymmetric play and education use cases.

#### Zoetone ‚Äî 3D Web Audio Visualizer
*MTR Magic Key LLP*  
Browser-based multi-channel music visualizer built with A-Frame WebXR + custom Web Audio pipeline.

- Links: [Project](https://www.mtrmagickey.com/zoetone) ¬∑ [Live demo](https://technobotanist.github.io/mtrmk-zoetone/) ¬∑ [Source](https://github.com/technobotanist/mtrmk-zoetone)

- Optimized for instant load on low-power hardware so schools can redeploy older machines.  
- Link: [Project page](https://www.mtrmagickey.com/mocking-board)
